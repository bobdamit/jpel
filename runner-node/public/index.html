This demo uses the following JPEL Runner API endpoints:

POST /api/processes - Load process definition
GET /api/processes - List loaded processes
POST /api/instances/:processId/start - Create new instance
POST /api/instances/:instanceId/step - Execute next step
POST /api/instances/:instanceId/activities/:activityId/submit - Submit human task
GET /api/instances/:instanceId/current-task - Get current human task
